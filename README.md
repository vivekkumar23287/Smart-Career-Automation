**📌 Project Overview**
The Smart Career Automation System is an AI-powered web platform designed to help students and job seekers automate their career development process. The system provides intelligent career guidance, learning resources, and automation tools to help users improve their skills and prepare for job opportunities.
The platform also integrates AI-based assistance and a secure payment gateway to unlock premium career resources and automation features.

**⚙️ Installation & Setup**
1️⃣ **Clone the repository**
git clone https://github.com/Vivek15911/Smart-Career-Automation-System.git
2️⃣ **Navigate to the project folder**
cd Smart-Career-Automation-System
3️⃣ **Install dependencies**
npm install
4️⃣ **Run the development server**
npm start

Application will run on:
http://localhost:3000
